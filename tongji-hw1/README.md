# 统计学习理论导论-实验课1

## 环境配置：
* Python3.7.4
* Pandas
* sklearn

## 文件结构：
```
  ├── Data.csv:数据集文件
  ├── perceptron.py: 感知机实现三分类
  ├── perceptron_frac.py: 训练集划分比例对精度的影响（出图用）
  ├── perceptron_lr.py: 学习率对精度的影响（出图用）
  ├── perceptron_seed.py: 权重初始化对精度的影响（出图用）
  ├── SVM.py: SVM实现三分类
  ├── SVM-c.py: 惩罚系数对精度的影响
  ├── SVM-gamma.py: rbf核函数gamma对精度的影响
  ├── SVM-kernel.py: 核函数对精度的影响
```
